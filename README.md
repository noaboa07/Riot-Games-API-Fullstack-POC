# League of Legends API - Proof of Concept (PoC)

## Overview
This repository contains a short sprint project from 2024 focused on integrating and exploring the official Riot Games API. The primary goal of this Proof of Concept was to practice handling asynchronous API calls, parsing complex JSON payloads, and extracting real-time player match data.

## Technical Focus
* **API Integration:** Establishing secure, real-time connections with the Riot Games REST API.
* **Data Parsing:** Extracting specific player statistics and match history from nested JSON structures.
* **Asynchronous Logic:** Implementing async/await patterns to manage data load times.

## Project Status
**Archived / Learning Exercise.** This project successfully served its purpose as an architectural foundation for API data fetching and frontend state management during my undergraduate studies. It is no longer being actively maintained.
